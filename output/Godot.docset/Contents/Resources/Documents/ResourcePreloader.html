ResourcePreloader

Brief Description

Resource Preloader Node.

Full Description

Resource Preloader Node. This node is used to preload sub-resources inside a scene, so when the scene is loaded all the resourcs are ready to use and be retrieved from here.

Methods

add_resource

add_resource ( String name , Object resource )
Add a resource to the preloader. Set the text-id that will be used to identify it (retrieve it/erase it/etc).

remove_resource

remove_resource ( String name )
Remove a resource from the preloader by text id.

rename_resource

rename_resource ( String name , String newname )
Rename a resource inside the preloader, from a text-id to a new text-id.

has_resource

bool has_resource ( String name )
Return true if the preloader has a given resource.

get_resource

Object get_resource ( String name )
Return the resource given a text-id.

get_resource_list

StringArray get_resource_list ( )
Return the list of resources inside the preloader.
